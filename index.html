Primitive and Non-Primitive Data Types

JavaScript has two main categories of data types:

Primitive Data Types

  • String: Represents text data, e.g., "Hello"
  • Number: Represents numeric values, e.g., 10, 3.14
  • Boolean: Represents true or false
  • Null: Represents a non-existent or invalid value
  • Undefined: Represents a variable that has not been assigned a value
  • Symbol: Represents a unique value for object properties (ES6+)
  • BigInt: Represents integers larger than 2^53 - 1 (ES11+)

Non-Primitive Data Types

  • Object: A collection of key-value pairs, e.g., { name: "John", age: 30 }
  • Array: A list-like object, e.g., [1, 2, 3, 4]
